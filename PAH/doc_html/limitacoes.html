Considerações e limitações sobre a aplicação

Verbatim

A tag PRE não é verbatim puro. Se dentro desta for colocado um ol, este será tratado como tal e não como mero texto. Esta é usada para verbatim de linha.
A tag xmp é utilizada para verbatim multi-linha (não pode ser usada para verbatim de linha porque força quebra de linha).

Link

O link não poderá conter a expressão inicial http://.

Caracteres

Assume-se que os todos os caracteres têm 1 byte de tamanho. Qual será o tamanho dos caracteres especiais (á, ã, â, etc)?

Output

Caso haja um argumento, o output será para um ficheiro com o nome especificado no argumento. Caso não haja, o output é o stdout.
